다음 시험 문제를 시스템에 넣을 JSON으로 변환해줘.

반드시 아래 규칙을 지켜줘:

JSON만 출력 (설명 절대 금지)
코드블록(```) 절대 사용 금지
마지막 줄은 반드시 } 로 끝나야 함
모든 문제 포함 (누락 금지)
answer는 반드시 채우기 (비워두면 안 됨)
explanation은 간단히 작성 (없으면 "")

형식:

{
"title": "시험 이름",
"grade": "중2",
"subject": "수학",
"teacher": "브라이언",
"difficulty": "기본",
"totalQuestions": 문제개수,
"questions": [
{
"no": 1,
"typeName": "유형",
"correctRate": 0,
"difficulty": "기본",
"answer": "정답",
"explanation": "해설"
}
]
}
